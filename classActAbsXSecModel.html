Detailed Description

Abstract class to define a model to calculate cross-sections.

All derived classes must implement the calcCrossSection method that calculates the cross-section for a given set of nuclei data (target, product) pairs.

Member Function Documentation

virtual double ActAbsXSecModel::calcCrossSection ( ActNucleiData *  data)
pure virtual

All sub-classes must implement the cross-section calculation. It takes the ActNucleiData pointer, which stores information about the target and product pair (Z,A) values, and uses this to calculate the cross-section. This should set the _sigma double private data member.

Implemented in ActXSecDataModel, ActSTPeripheral, ActSTSLite, ActSTEvaporation, ActSTSpallation, ActSTBreakup, ActSTFissBreakup, ActSTFission, ActSTFissSpallation, ActSTFissSpallGamma, and ActTritiumModel.

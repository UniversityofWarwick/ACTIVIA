Detailed Description

Use data tables to calculate cross-sections.

This class allows us to use experimental data to obtain cross-sections for target-product isotope pairs. Each individual table must be for only one target isotope but can include any number of product isotopes. The files must be written in a common format to be interpreted and used correctly. The start of the file can contain comment lines each preceeded by the hash (#) symbol.

  1. The target isotope is specified by a line
    Target Zt At
    where Zt and At are the atomic and mass numbers of the isotope.
  2. Next another comment line should be written - this specifies the start of a table for a product isotope, which itself must be defined on the following line
    # New table comment
    Product Z A
  3. The following lines must then contain the values of energy (MeV) and cross-section (mb)
    10.0 0.1
    20.0 0.25
    ... ...
    A new product table can be written after the proceeding table by repeating steps 2 and 3 above.

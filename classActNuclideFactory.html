Detailed Description

A factory class for creating nuclear isotope objects.

This class must be used to create nuclear isotopes. It uses the factory method whereby if the required isotope already exists it returns the pre-existing pointer and does not create a new object. This significantly saves on memory use, and avoids the need to continually create new isotope objects whenever they are used throughout the code.
